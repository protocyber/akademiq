# School Admin Guide

Documentation for school administrators managing academic and system configuration.

## Managing semester evaluations

Open **Settings → Academic → Terms**, edit a semester, then use the **Evaluasi** tab after configuring report types in **Rapor**.

Use the evaluation list to add semester-wide columns such as `UH1`, `UTS`, and `UAS`. Use the weight matrix to set each report type's weights; every report type column must total 100% before saving.

When teaching assignments already exist, click **Terapkan ke semua penugasan yang belum punya evaluasi**. The action fills only assignments that still have no evaluations and is safe to run repeatedly. A banner shows how many assignments still need evaluation materialization.

The semester template is only a starting point. Teachers can still adjust concrete evaluations for their own class and subject after the template has been applied.
